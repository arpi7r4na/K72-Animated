# 🎬 K72 Animated Website

A modern, animation-focused agency website inspired by K72, built to deliver smooth scrolling experiences, immersive motion design, and premium UI interactions.

🔗 **Live Demo:**  
https://k72-arp1t.onrender.com

---

## ✨ About The Project

K72 Animated is a front-end project built to replicate a high-end creative agency experience. The website focuses heavily on smooth scroll animations, video integration, and performance-optimized transitions using modern web technologies.

This project highlights advanced animation control, production-ready structuring, and responsive design principles.

---

## 🚀 Tech Stack

- React  
- Vite  
- GSAP  
- ScrollTrigger  
- Tailwind CSS  
- JavaScript (ES6+)  

---

## 🌟 Features

- Smooth scroll-based animations  
- GSAP timeline & ScrollTrigger effects  
- Background video integration  
- Fully responsive layout  
- Optimized production build  
- Clean and minimal UI design  

---

## 🛠 Installation & Setup

Clone the repository:

git clone https://github.com/arpi7r4na/K72-Animated.git  

Navigate to the project directory:

cd k72-main  

Install dependencies:

npm install  

Run development server:

npm run dev  

Build for production:

npm run build
